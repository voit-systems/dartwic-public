DARTWIC 1.0 starts with three engine license tiers.

# Free

- Up to 3 module instances
- Up to 250 channels

# Pro

- Up to 25 module instances
- Up to 5,000 channels

# Enterprise

- Unlimited by default policy

# Enforcement

Tier limits are enforced directly in the engine. If a configuration exceeds the licensed module or channel capacity, the engine blocks that usage with a clear error instead of silently degrading.
